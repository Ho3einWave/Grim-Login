# Installation:

1. Run `install.bat`

2. Run `run.bat`

OR


1. `pip install -r requirements.txt`

2. `python main.py`


### Python:

Python 3.6 or higher are compatible, you can get them here:

[Python](https://www.python.org/downloads/)
